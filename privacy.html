Privacy Policy

This policy describes the data practices of the current static version of GTA Dryer Vent Guide.

Current quote-form status

The quote form is currently in preview mode. It does not transmit or store the information entered into it, and no quote request is sent to a provider. This policy will be updated if an intake service is enabled.

Technical information

The site is hosted through GitHub Pages. GitHub and infrastructure providers may process technical information needed to deliver and secure the website, such as IP address, browser type, device type, requested pages and timestamps. Their handling is governed by their own privacy terms.

Cookies and analytics

This website does not currently include advertising cookies, analytics, call tracking or remarketing code. If tracking technology is added, this section and any required consent controls will be updated before that technology is used.

How future quote intake will work

If quote intake is enabled, the form will identify what information is collected, the purpose of collection and whether an opted-in request may be shared with a participating local provider. The provider will have its own privacy practices.

Retention

Because the current quote form does not transmit entries, this website does not retain quote-request data. Infrastructure providers may retain technical logs under their own policies.

Security

The website is delivered over HTTPS. No online service can guarantee absolute security.

Children

This website is intended for adults researching or arranging property services, not children.

Changes

We may update this policy when the site’s tools or data practices change. The current revision date appears at the top of this page.
